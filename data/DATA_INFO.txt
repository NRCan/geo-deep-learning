The data provided in massachisetts_buildings.zip is a subset of the "Massachusetts Buildings Dataset" introduced in the PhD thesis of
Mnih V. Machine learning for aerial image labeling. University of Toronto (Canada), 2013.
The thesis can be found here: https://www.cs.toronto.edu/~vmnih/docs/Mnih_Volodymyr_PhD_Thesis.pdf

There is no licence associated to this data, as stated in the thesis.
The images have been cropped from their original source and the vector data has been modified, in order to reduce the size of the dataset.

The rest of the dataset can be found at this url: https://www.cs.toronto.edu/~vmnih/data/

The data provided in classification_data.zip is a subset of the "DeepSat SAT6" dataset (Source: Saikat Basu, Sangram Ganguly, Supratik Mukhopadhyay, Robert Dibiano, Manohar Karki and Ramakrishna Nemani, DeepSat - A Learning framework for Satellite Imagery, ACM SIGSPATIAL 2015). 

The images provided have been converted from their original csv format to Tiff images. The rest of the dataset can be downloaded from https://www.kaggle.com/crawford/deepsat-sat6/home in .csv or .mat file formats.

More information on this dataset can be found at this URL: https://csc.lsu.edu/~saikat/deepsat/
